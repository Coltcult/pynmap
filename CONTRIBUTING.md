# 🤝 Contributing to PyNmap

First off, thank you for considering contributing to PyNmap!

## 📜 Code of Conduct
By participating, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

## 🎯 How Can I Contribute?

### 🐛 Report Bugs
- Check if the bug already exists in [Issues](https://github.com/Coltcult/pynmap/issues)
- Create a new issue with clear steps to reproduce

### 💡 Suggest Enhancements
- Open an issue with the label `enhancement`
- Describe the feature and why it is valuable

### 🔧 Add New Vulnerability Scripts
1. Create a new .py file in scripts/
2. Follow the template in the README
3. Test your script before submitting

### 📝 Improve Documentation
- Fix typos or unclear explanations
- Add examples or use cases

## 🛠️ Development Setup

bash:
# Clone the repository
git clone https://github.com/Coltcult/pynmap.git
cd pynmap

# Install dependencies
pip install -r requirements.txt

## 📥 Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Push to your fork
5. Open a Pull Request

## 🎨 Style Guidelines
- Follow PEP 8
- Add docstrings to all functions
- Use descriptive variable names

Thank you for contributing!
