## 📌 Overview

**PyNmap** is an advanced, intelligent network scanner that leverages **Reinforcement Learning** and **Payload Evolution** to detect vulnerabilities with unprecedented accuracy. Designed for security researchers and CTF players, it's a powerful tool that operates without root privileges.

## ✨ Key Features

- 🧠 **Reinforcement Learning** - Adapts scanning strategies based on network responses
- 🔄 **Payload Evolution** - Dynamically generates and mutates payloads for better detection
- 🛡️ **15+ Vulnerability Scripts** - Detects SQLi, XSS, LFI, and more
- 🏴 **CTF Mode** - Automatic flag detection and capture
- 🚀 **No Root Required** - Runs safely with user privileges
- 📊 **Smart Reporting** - JSON/HTML output with detailed vulnerability analysis
