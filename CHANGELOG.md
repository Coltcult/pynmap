# 📝 Changelog

All notable changes to PyNmap will be documented in this file.

## [2.0.0] - 2024-05-22

### Major Features
- Added Reinforcement Learning module (pynmap_learning.py)
- CTF Mode with automatic flag detection
- 15+ vulnerability detection scripts
- Payload evolution system

### Enhancements
- Improved scan speed with threading
- JSON and HTML report generation
- Colored terminal output
- Progress bars for long scans

### Bug Fixes
- Fixed false positives in SQL injection detection
- Improved error handling for network timeouts

## [1.0.0] - 2024-03-15

### Initial Release
- Basic port scanning
- 5 vulnerability detection scripts
- Simple reporting
