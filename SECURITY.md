# 🔒 Security Policy

## Supported Versions
Currently supported versions with security updates:
- 2.x: Fully supported
- 1.x: No longer supported

## Reporting a Vulnerability
If you discover a security vulnerability:
1. DO NOT open a public issue
2. Email us at security@pynmap.com
3. Include details about the vulnerability and steps to reproduce

We will respond within 48 hours and coordinate a fix before public disclosure.

## Security Best Practices
- Only scan targets you own or have permission to test
- Use in isolated environments when testing unknown targets
- Keep your installation updated
- Review community scripts before running them
